# 🎲 Dice Roller

A simple Dice Roller web application built using **HTML, CSS, and JavaScript**. This project allows users to roll one or multiple dice and displays random results using dice symbols.

---

## 🚀 Features

* Roll any number of dice (minimum 1)
* Random results from 1 to 6
* Clean and responsive UI
* Dice displayed using Unicode symbols (⚀ ⚁ ⚂ ⚃ ⚄ ⚅)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📁 Project Structure

```
Dice Roller Project/
│
├── index.html
├── style.css
└── index.js
```

---

## 🎮 How It Works

1. Enter the number of dice you want to roll.
2. Click the **Roll Dice!** button.
3. The result will appear instantly using dice symbols.

---

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Start rolling dice!
